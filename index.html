Welcome to the PHYGITAL CLAYING SYSTEM!
This is a Visual based co-creative system for generating 3D sculptures.

Always wanted to see objects in the world in another way?

Then please position your camera on an object with a plain background!

The camera view is converted through the OpenCV edge detection.
The creative System uses this input and tries to see depth by having a point & mutation
system on the details. It is then put in a 3D array that can be saved in an SCV.
